## Code Example
### Task: 
Count the number of divisors of a positive integer `n`.
### Solution: 
```
int divisors(int n){  
  int j = 1;
  for(int i = 2; i <= n; i++){
    if(n % i == 0) j++;
  }
  return j;
}
```
